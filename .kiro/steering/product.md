# プロダクト概要

Water Sort Puzzle Solver は、ウォーターソートパズル（カラーウォーターソーティングゲーム）の初期状態をファイルで受け取り、有効な解法手順を自動探索して出力する Python 製 CLI ツールである。

## コア機能

- **パズル入力の解析**: YAML / JSON / テキスト形式のファイルからボトル配色を読み込む
- **状態検証**: 入力の整合性（各色セグメント数の倍数チェック、ボトル数範囲）を確認する
- **自動解法探索**: BFS（最短手数）または DFS（高速）でパズルを解く
- **柔軟な出力**: テキスト・JSON・YAML 形式で解法ステップを出力する

## 対象ユースケース

- ゲームプレイヤーが詰まったパズルの解法を取得する
- `--validate` フラグのみで入力ファイルの妥当性をチェックする

## 価値提案

手動では解けない難易度のパズルを、単一コマンドで解決できる点が価値。BFS / DFS の選択により速度と最適性のトレードオフを制御できる。
